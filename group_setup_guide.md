# 그룹 채팅에서 AI 봇 설정하기

## 🔧 BotFather에서 Privacy 설정 변경

### 1단계: BotFather에게 메시지 보내기
```
/mybots
```

### 2단계: Bot 선택
**@AI_Solarbot** 선택

### 3단계: Bot Settings 클릭
- Bot Settings 선택

### 4단계: Group Privacy 선택
- **Group Privacy** 클릭

### 5단계: Turn OFF 선택
- **Turn OFF** 선택
- 확인 메시지에 **Confirm** 클릭

## 📱 그룹에 Bot 추가

### 1단계: 그룹에 Bot 초대
- 그룹 채팅에서 **@AI_Solarbot** 검색
- **추가** 버튼 클릭

### 2단계: Bot 권한 설정
- **Administrator** 권한 부여
- 또는 최소 **Send Messages** 권한

## 🧪 테스트

그룹에서:
1. @AI_Solarbot 멘션하거나
2. 일반 텍스트 보내기
3._bot이 응답하는지 확인

## ⚠️ 주의사항

Bot이 그룹에서 작동하지 않는 경우:
1. Privacy Mode가 여전히 ON이면 OFF로 변경
2. Bot이 Administrator 권한이 없으면 권한 부여
3. 다른 그룹에서 테스트해보기
